# Навык: Sync to Global (Синхронизация с Глобальным Мозгом)

**Когда использовать:**
1. Когда пользователь генерирует или добавляет общедоступную (public) полезную информацию.
2. Когда пользователь явно просит "отправь это в глобальную базу", "поделись этим с другими министерствами".
3. **НИКОГДА** не используй этот навык для документов с грифом "секретно", "конфиденциально" или "внутреннее".

**Как использовать:**
Вызови Python скрипт для отправки:
`python config/skills/sync_to_global/scripts/sync.py "текст или выжимка знаний для синхронизации" --ministry "min_energo"`

Скрипт автоматически векторизует текст и отправит его на Глобальный Сервер с тегом `security_level: public`.
