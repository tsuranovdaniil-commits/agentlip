# Навык: Multi-Vector Search (Глобальный + Локальный поиск)

**Когда использовать:**
1. Если пользователь задает масштабный вопрос: "Какие вообще есть данные по X?", "Что пишут другие министерства о Y?".
2. Если ответ не найден в обычном контексте или локальной памяти агента, и нужно сделать глубокий федеративный поиск.

**Как использовать:**
Вызови Python скрипт для поиска:
`python config/skills/multi_vector_search/scripts/search.py "твой поисковый запрос" --ministry "min_energo"`

Скрипт сделает запросы в оба Qdrant-инстанса и вернет объединенный контекст (с учетом прав доступа).
Используй полученный результат для формирования ответа пользователю.
